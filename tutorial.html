Instalasi & Penggunaan
iMersTeleship

Selamat datang di ekosistem E-Commerce Telegram otomatis! Ikuti panduan langkah demi langkah di bawah ini. Tidak perlu keahlian coding, toko Anda akan Live dalam 10 menit!

1 Persiapan "Kunci" Toko Anda

A Dapatkan Token Bot Anda

  • Buka aplikasi Telegram, cari bot bernama @BotFather.
  • Ketik /newbot lalu ikuti instruksi untuk membuat Nama dan Username bot Anda.
  • BotFather akan memberikan HTTP API Token (contoh: 1234567:ABCdefGHI...).
  • Copy dan simpan Token tersebut!

B Dapatkan Admin Chat ID Anda

  • Buka Telegram, cari bot bernama @userinfobot (atau bot sejenis).
  • Klik Start. Bot akan membalas dengan info Anda, termasuk Id (contoh: 987654321).
  • Copy dan simpan Angka tersebut. Ini berguna agar laporan orderan masuk ke Telegram pribadi Anda.

2 Salin Database & Deploy Server

Sekarang kita akan menyalin sistem ke akun Google Anda agar toko ini sepenuhnya menjadi milik Anda.

  1. Salin File Template: Buka link Google Sheets yang kami berikan, lalu klik File -> Buat Salinan (Make a copy). Simpan di Google Drive Anda.
  2. Setup Database Awal:
    Di file salinan Anda, perhatikan menu atas. Klik menu 🚀 iMersTeleship App lalu pilih ⚙️ Setup Database Awal.
    (Jika Google meminta izin keamanan, klik Lanjutkan -> Pilih Email Anda -> Advanced -> Go to ... (Unsafe) -> Allow).
  3. Buka Dapur Sistem: Klik menu Ekstensi (Extensions) -> Apps Script. Tab kodingan akan terbuka.
  4. Deploy (Terapkan) Server:
    • Di sudut kanan atas Apps Script, klik tombol biru Terapkan (Deploy) -> Penerapan Baru (New Deployment).
    • Klik ikon gerigi (⚙️) di sebelah kiri, centang Aplikasi Web (Web App).
    • Atur persis seperti ini:
      - Deskripsi: Versi 1
      - Execute as: SAYA (Me)
      - Who has access: Siapa saja (Anyone)
    • Klik Terapkan (Deploy). Akan muncul URL Aplikasi Web (berakhiran /exec). COPY URL TERSEBUT!

3 Aktifkan Bot (1 Klik!)

Ini adalah keunggulan iMersTeleship. Anda tidak perlu pusing mencari pengaturan koding rahasia, kami sudah menyediakan UI Admin khusus!

  1. Kembali ke tab Google Sheets Anda.
  2. Klik menu 🚀 iMersTeleship App lalu pilih 💻 Buka Dashboard Admin.
  3. Akan muncul panel kontrol di sisi kanan layar Anda. Buka tab 🔑 API.
  4. Masukkan 3 data yang sudah Anda kumpulkan: Bot Token, Admin Chat ID, dan Web App URL.
  5. Klik Simpan & Aktifkan.
Selesai! Jika muncul notifikasi "Tersimpan!", berarti Bot Anda sudah 100% tersambung dan nyala.

4 Manajemen Toko

Tambah Produk

  • Buka menu 🚀 iMersTeleship App -> 💻 Buka Dashboard Admin.
  • Pilih tab 📦 Produk. Isi formulir (Nama, Harga, dll).
  • Pada kolom Link Akses / GDrive, masukkan link rahasia (file/grup) Anda. Produk akan otomatis tampil di bot Telegram.

Kelola Pesanan

  • Ketik perintah /admin di Bot Telegram Anda sendiri.
  • Pilih Cek Order Pending.
  • Lakukan verifikasi, lalu klik Approve untuk mengizinkan akses atau Reject untuk menolak pesanan secara manual.

5 Bedah Fitur Custom Menu (Add-on)

Di bagian atas Google Sheets Anda terdapat menu khusus bernama 🚀 iMersTeleship App. Berikut adalah fungsi dari tiap fiturnya:

💻 Buka Dashboard Admin

Membuka panel modern di sisi kanan layar untuk mengatur koneksi API, input Produk Baru, Rekening Bank, dan Teks Balasan (Copywriting) dengan sangat mudah.

🤖 Uji Coba: Ping Bot & 🛍️ Katalog Produk

Fitur "Tes Ombak". Memaksa bot mengirim pesan atau katalog langsung ke Telegram Anda (Admin) untuk memastikan sistem berjalan normal sebelum dirilis ke publik.

📡 Diagnosa: Cek Status Webhook

Alat troubleshooting untuk mendeteksi error koneksi. Akan menampilkan status apakah server Google berhasil terhubung dengan server Telegram.

⚙️ Setup Database Awal & 📝 Masukkan Data Dummy

Digunakan hanya 1x saat awal instalasi. Berfungsi membuat struktur tabel (Products, Orders, dll) yang rapi dan mengisi contoh data agar Anda bisa langsung mengetes sistem.

🔄 Reset & Bersihkan Antrean Bot

Fitur darurat andalan! Gunakan tombol ini jika bot tiba-tiba tidak merespon, terasa sangat lambat (lag), atau terjadi penumpukan pesan. Sistem akan menghapus antrean yang macet di Telegram dan menyegarkan kembali koneksi bot Anda secara instan.
